# Древо рода Житниковых

Иммерсивный сайт семейного древа: живое 3D-древо рода (three.js),
неоновая карта расселения, рабочее 2D-древо, галерея и хроника.
572 человека, 8 поколений, 1858—2026. Данные — экспорт MyHeritage (GEDCOM).

- `docs/` — сайт (статический, без сборки; three.js лежит в `docs/vendor`)
- `docs/js/drevo3d.js` — 3D-древо: ветвь — человек, лист — жизнь
- `docs/js/map3d.js` — карта мест рождений и миграций (геокодинг Nominatim)
- `data/parse.py` — конвертер GEDCOM → `docs/data.json`
- `data/geocode.py` — геокодинг мест → `docs/places.json`

Локально: `python3 serve.py 8643`
